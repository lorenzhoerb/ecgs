TLDR:

One Important Part of a well made Frontend is Localization for the most important Keywords. Therefore I added an LocalizationService class to help with that matter.

How to use it

The correct Usage is demonstrated on this UIDemo Page. The LocalizationService takes advantage of Typescripts typing system and is fully typed itself. To localize a key just call localize.WhatYouWantToLoacalize. If a key does not yet exist you need to add it in `language.ts` to the `LocalizationKeys`class. Afterwards you have to add a valid translation to each language file we support or the frontend won't compile. Keys come in two forms:
  • Simple keys that just represent constants like "{{localize.Hello}}"
  • Complex keys that evaluate functions like "{{localize.HelloFunction(1)}}" which can also create: "{{localize.HelloFunction(3)}}"
